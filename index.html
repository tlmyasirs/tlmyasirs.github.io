IoT for Home Automation

Designed a Home automation system with IoT. The design consists of a Raspberry Pi 2 model B, 4 Arduino Nano boards, 5 nRF24L01 wireless modules, sensors, relays, other essential electronic components, and the Adafruit IoT platform via io.adafruit.com.

The Raspberry Piâ€™s SPI interface connects the nRF24L01 wireless transceiver module. Sensors and relays connected to Arduino slaves communicate wirelessly via nRF24L01.

Home Automation is the future of smart living.

IFTTT and MQTT Protocols

A machine-to-machine (M2M) protocol, MQTT is used for communication. Additionally, IFTTT is integrated for enhanced automation and connectivity between various services.

Programming and Data Flow

Using MQTT client library, I wrote Python code to receive sensor data and feed it to the IoT platform. Testing was done using MQTT.fx, and the data successfully reached the IoT platform.

The slaves are controlled using C++ to handle the wireless module nRF24L01, while Python code feeds data to Adafruit IO, as shown in the architecture diagram.
